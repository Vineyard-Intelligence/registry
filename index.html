Vineyard registry

The catalog index for Vineyard Type Packs and Plugin Packs.

Published endpoints GET /registry/community-typepacks.json GET /registry/community-plugins.json
Add a pack

Open a pull request adding an entry to registry/community-typepacks.json or registry/community-plugins.json pointing at your content repo and release tag. Pack files themselves live in their own repositories.

Documentation: docs.vineyard.run · Source: github.com/Vineyard-Intelligence/registry

The documentation site and the in-app marketplace render this catalog over XHR. This registry stores metadata only — it never hosts or executes plugin code.
